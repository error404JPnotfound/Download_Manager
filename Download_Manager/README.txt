QUICK START (Windows)
──────────────────────
1. Unzip this archive.
2. Double-click  RocketDL.exe
   • No installation required.
   • Config is stored in  %APPDATA%\RocketDL\

QUICK START (Kali / Debian Linux)
──────────────────────────────────
1. Install the package:
     sudo dpkg -i rocketdl_1.0_amd64.deb
     sudo apt-get install -f          # fix any missing deps
2. Launch from the application menu or run:
     rocketdl

SYSTEM REQUIREMENTS
────────────────────
Windows : Windows 10/11 (64-bit). No extra software needed.
Linux   : Debian / Kali Linux (amd64). Python 3.10+ required.
          Run  sudo apt-get install -f  after dpkg install.

BUILDING FROM SOURCE
─────────────────────
Requirements: Python 3.10+, pip install pywebview pyinstaller yt-dlp nodriver

Windows:
  python -m PyInstaller --clean RocketDL.spec

Linux:
  pip install pywebview yt-dlp nodriver
  python app.py
